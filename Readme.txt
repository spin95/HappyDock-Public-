HappyDock
------------------------------

Description :

Ce projet consiste a permettre facilement la mise en place :
	- D'un envirronnement de développement pour ReactJs
	- D'un envirronnement de développement pour Wordpress
	- D'un envirronnement pour déployer un projet développer sous LAMP.


------------------------------

Pré requis : 

	- Installation de docker : https://www.docker.com/
	- Copie des fichiers du projet LAMP depuis HappyDock/lamp/src vers C:\Users\[nom d'utilisateur]\www


------------------------------

Démarrer : 

	- Lancer une Invite de Commande ou le Powershell Windows, se placer dans le dossier HappyDock,
	  puis démarrer avec "docker-compose up -d".


------------------------------

Arreter: 

	- Sur la même Invite de commmande stopper avec "docker-compose stop".
